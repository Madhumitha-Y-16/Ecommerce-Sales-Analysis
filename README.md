# 📊 E-commerce Sales & Profitability Analysis

## 🔍 Overview
This project analyzes e-commerce sales data to identify key drivers of revenue and profitability using SQL and Power BI.

## 🛠 Tools Used
- SQL
- Power BI


## 📈 Analysis Performed
- Profit trend analysis over time
- Discount impact on profitability
- Category and regional performance analysis
- Customer concentration analysis
- Loss detection using heatmap

## 🔑 Key Insights
- High discounts reduce profitability
- Certain categories have low margins
- Losses are concentrated in specific regions
- Revenue depends on top customers
